# NAME

load-rdm-records.bash

# SYNOPSIS

load-rdm-records.bash REPO_ID [YYYY-MM-DD]

# DESCRIPTION

Load a snapshot of an RDM database replacing the existing database
if necessary.

# EXAMPLE

~~~
    load-rdm-records.bash caltechdata 2022-10-20
~~~
